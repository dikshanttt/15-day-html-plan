What is HTML?

HTML (HyperText Markup Language) is the foundation of every website. It's like the skeleton of a webpage — it gives structure and meaning to content.

Think of it this way:

  • HTML = Structure (like the bones of a body)
  • CSS = Styling (like clothes and makeup)
  • JavaScript = Interactions (like movements and actions)

Essential HTML Tags to Know

<html> - The root tag that wraps everything
<head> - Contains metadata and links to stylesheets
<body> - Contains all visible content
<h1> to <h6> - Headings (h1 is largest, h6 is smallest)
<p> - Paragraph text
<div> - A container to group elements

Setup: Install VS Code

  1. Visit code.visualstudio.com
  2. Download for your operating system
  3. Install and open the editor
  4. Install extensions:
    • Live Server (by Ritwick Dey)
    • Prettier - Code formatter
    • HTML Preview

Your First HTML Page

Create a file called index.html and write:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is my first webpage!</p>
</body>
</html>

Set Up GitHub

  1. Go to github.com and create a free account
  2. Create a new repository called "my-first-website"
  3. Upload your index.html file
  4. Enable GitHub Pages to view your site live
