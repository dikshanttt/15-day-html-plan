What is Flexbox?

Flexbox is a modern CSS layout method that makes arranging elements incredibly easy. It automatically spaces and aligns items for you.

Instead of fighting with positioning, Flexbox:

  • Centers items perfectly
  • Distributes space automatically
  • Reflows content responsively
  • Simplifies complicated layouts

Basic Flexbox Setup

To use Flexbox, add display: flex; to a container:

.container {
  display: flex;
}

All direct children automatically become flexible items!

Key Flexbox Properties

justify-content: Align items horizontally
align-items: Align items vertically
flex-direction: row or column
gap: Space between items
flex-wrap: Wrap items to next line

Justify Content Values

justify-content: flex-start;    /* Left aligned */
justify-content: center;        /* Centered */
justify-content: flex-end;      /* Right aligned */
justify-content: space-between; /* Space between items */
justify-content: space-around;  /* Space around items */
justify-content: space-evenly;  /* Even spacing */

Complete Flexbox Example

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 20px;
  background-color: #333;
  padding: 10px 20px;
}

.nav-items {
  display: flex;
  gap: 15px;
}

.card-container {
  display: flex;
  gap: 20px;
  flex-wrap: wrap;
  justify-content: center;
}

Bootstrap Grid Example

Bootstrap makes responsive layout easier with its grid system. It uses rows and columns so content adjusts automatically.

<div class="container">
  <div class="row g-3">
    <div class="col-12 col-md-6">Item 1</div>
    <div class="col-12 col-md-6">Item 2</div>
  </div>
</div>

Center Anything with Flexbox

To perfectly center content horizontally AND vertically:

.center {
  display: flex;
  justify-content: center;  /* Horizontal */
  align-items: center;     /* Vertical */
  height: 300px;            /* Set height */
}
