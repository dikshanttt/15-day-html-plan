What is the DOM?

The DOM (Document Object Model) is how JavaScript sees your HTML page. It's like a tree of all your elements.

JavaScript can:

  • Find elements (select them)
  • Change their content
  • Change their styles
  • Add/remove classes
  • Create new elements
  • Delete elements

Selecting Elements

// Get by ID
let element = document.getElementById("myId");

// Get by class (returns first match)
let element = document.querySelector(".myClass");

// Get all elements with a class
let elements = document.querySelectorAll(".myClass");

// Using CSS selectors
let element = document.querySelector("div > p.highlight");

Modifying Text Content

let para = document.querySelector("p");

// Change text
para.textContent = "New text content";

// Change HTML
para.innerHTML = "<strong>Bold text</strong>";

Note: Use textContent for plain text, innerHTML for HTML.

Adding/Removing Classes

let button = document.querySelector("#myBtn");

// Add a class
button.classList.add("active");

// Remove a class
button.classList.remove("active");

// Toggle a class
button.classList.toggle("active");

// Check if has class
if (button.classList.contains("active")) {
  console.log("Button is active");
}

Changing Styles

let div = document.querySelector("div");

div.style.color = "blue";
div.style.backgroundColor = "yellow";
div.style.fontSize = "18px";
div.style.display = "none"; // Hide element
div.style.display = "block"; // Show element

CSS property names become camelCase in JavaScript:

  • background-color → backgroundColor
  • font-size → fontSize

Dark Mode Toggle Example

<style>
  .dark-mode {
    background-color: #1a1a1a;
    color: white;
  }
</style>

<button id="themeToggle">Toggle Dark Mode</button>

<script>
let toggleBtn = document.getElementById("themeToggle");
let body = document.body;

toggleBtn.addEventListener("click", function() {
  body.classList.toggle("dark-mode");
});
</script>

Show/Hide Elements

let hideBtn = document.querySelector("#hideBtn");
let content = document.querySelector(".content");

hideBtn.addEventListener("click", function() {
  if (content.style.display === "none") {
    content.style.display = "block";
    hideBtn.textContent = "Hide";
  } else {
    content.style.display = "none";
    hideBtn.textContent = "Show";
  }
});
