Day 12: Events and Interactions

Making Things Happen

What You'll Learn Today

  • Respond to user actions (clicks, hovers)
  • Use onclick and addEventListener
  • Change content dynamically
  • Create button interactions

What are Events?

Events are things users do: clicking buttons, typing in forms, moving the mouse, etc.

Your JavaScript can listen for these events and respond!

Common Events

click - User clicks an element
mouseover - Mouse hovers over element
mouseout - Mouse leaves element
change - Form field value changes
submit - Form is submitted
keypress - User presses a key

Method 1: onclick in HTML

<button onclick="sayHello()">Click Me</button>

<script>
function sayHello() {
  alert("Hello!");
}
</script>

Method 2: addEventListener (Recommended)

<button id="myBtn">Click Me</button>

<script>
let button = document.getElementById("myBtn");

button.addEventListener("click", function() {
  alert("Button was clicked!");
});
</script>

Change HTML Content

<p id="message">Original text</p>
<button id="changeBtn">Change Text</button>

<script>
let btn = document.getElementById("changeBtn");
let msg = document.getElementById("message");

btn.addEventListener("click", function() {
  msg.textContent = "Text has been changed!";
});
</script>

Change CSS with JavaScript

<div id="box">Click me</div>

<script>
let box = document.getElementById("box");

box.addEventListener("click", function() {
  box.style.backgroundColor = "blue";
  box.style.color = "white";
});
</script>

Toggle Classes

<style>
  .active {
    background-color: blue;
    color: white;
  }
</style>

<button id="toggleBtn">Toggle</button>

<script>
let btn = document.getElementById("toggleBtn");

btn.addEventListener("click", function() {
  btn.classList.toggle("active");
});
</script>
