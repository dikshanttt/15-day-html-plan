Day 4: HTML Forms

Collecting User Input

What You'll Learn Today

  • Create forms to collect user input
  • Use input, textarea, and button elements
  • Add labels for accessibility
  • Build checkboxes and radio buttons

The Form Tag

Forms group input elements together:

<form>
  <!-- Form elements go here -->
</form>

Input Types

<input type="text"> - Single line text
<input type="email"> - Email input
<input type="password"> - Hidden text
<input type="number"> - Numbers only
<input type="checkbox"> - Multiple selections
<input type="radio"> - Single selection
<textarea> - Multi-line text

Labels and Accessibility

Always use <label> to describe inputs:

<label for="name">Your Name</label>
<input type="text" id="name">

The for attribute connects the label to the input using the input's id.

Complete Form Example

<form>
  <label for="name">Name</label>
  <input type="text" id="name" name="name">

  <label for="email">Email</label>
  <input type="email" id="email" name="email">

  <label>
    <input type="checkbox"> I agree to terms
  </label>

  <label>Choose one:
    <input type="radio" name="choice"> Option A
    <input type="radio" name="choice"> Option B
  </label>

  <label for="message">Message</label>
  <textarea id="message" name="message"></textarea>

  <button type="submit">Submit</button>
</form>

Important Attributes

  • name - Identifies the field when form is submitted
  • id - Unique identifier for the element
  • placeholder - Hint text inside the input
  • required - Field must be filled before submitting
  • value - Default value for the input

Your Task for Today

Create a contact form with:

  • Name input field
  • Email input field
  • A textarea for messages
  • Checkboxes for preferences
  • A submit button
  • Proper labels for all fields
