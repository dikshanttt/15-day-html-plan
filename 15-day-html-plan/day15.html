Day 15: Build Your Final Project

Everything Together

What You'll Learn Today

  • Combine HTML, CSS, and JavaScript
  • Build a complete, functional website
  • Deploy your project to GitHub Pages

Project Ideas

Choose one of these projects to build:

  • Recipe Website: Display recipes with ingredients, instructions, and a search function
  • Portfolio Site: Showcase your projects with filters and descriptions
  • Todo App: Add, delete, and complete tasks with local storage
  • Product Catalog: Display products with search, filter, and shopping cart
  • Blog: Display blog posts with categories and comments

What Your Project Should Include

  • HTML: Semantic structure with proper tags
  • CSS: Responsive design that works on mobile and desktop
  • JavaScript: At least 2 interactive features (buttons, forms, dynamic content)
  • Design: Clean, professional appearance with consistent colors
  • Navigation: Clear menu and way to navigate between pages

Final Checklist

✓ All files organized in folders (css/, images/, pages/)
✓ Responsive design works on phones and tablets
✓ At least one interactive JavaScript feature
✓ No console errors (check F12)
✓ Good color scheme (3-5 colors max)
✓ Proper spacing and typography
✓ Links work correctly
✓ Images load properly

Deploy to GitHub Pages

  1. Push your project to GitHub
  2. Go to repository → Settings
  3. Find "GitHub Pages" section
  4. Select main branch as source
  5. Your site will be live at: yourusername.github.io/repo-name

Example Project: Recipe Website

HTML: Header, navigation, recipe cards, search form

CSS: Card layout, responsive grid, beautiful typography

JavaScript: Search recipes, filter by category, theme toggle

Performance Tips

  • Compress images (use tools like TinyPNG)
  • Minimize CSS and JavaScript files
  • Use external stylesheets instead of inline styles
  • Load scripts at the end of body tag
  • Test on real devices, not just browser
