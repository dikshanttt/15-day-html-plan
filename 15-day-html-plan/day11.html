Day 11: JavaScript Basics

Adding Interactivity

What You'll Learn Today

  • What JavaScript does
  • Variables and data types
  • Writing functions
  • Using console.log() for debugging

What is JavaScript?

JavaScript makes websites interactive. It's the programming language of the web.

  • HTML = Structure
  • CSS = Appearance
  • JavaScript = Behavior & Interactivity

Adding JavaScript to HTML

Add a script tag before the closing </body> tag:

<script>
  console.log("Hello World!");
</script>

Or link to an external file:

<script src="script.js"></script>

Variables

Variables store data that your program can use and change:

let name = "John";
let age = 25;
let isStudent = true;

console.log(name); // Prints: John
console.log(age); // Prints: 25

Use let or const to declare variables (never use var).

Data Types

String: Text in quotes - "Hello"
Number: 42, 3.14, -5
Boolean: true or false
Array: [1, 2, 3, "text"]
Object: {name: "John", age: 25}

Basic Math Operations

let x = 10;
let y = 3;

console.log(x + y); // 13 (addition)
console.log(x - y); // 7 (subtraction)
console.log(x * y); // 30 (multiplication)
console.log(x / y); // 3.33 (division)
console.log(x % y); // 1 (remainder)

Functions

Functions are reusable blocks of code:

function greet(name) {
  console.log("Hello " + name);
}

greet("Alice"); // Prints: Hello Alice

function add(a, b) {
  return a + b;
}

let result = add(5, 3);
console.log(result); // Prints: 8

Using console.log()

console.log() prints messages to the browser console. It's your best debugging tool!

console.log("A string");
console.log(42);
console.log(true);
console.log({name: "John", age: 25});

// To open DevTools: F12 or Right-click → Inspect → Console tab
