Day 2: Images, Links, and Lists

Adding Rich Content

What You'll Learn Today

  • Add images with the <img> tag
  • Create links with the <a> tag
  • Build lists (ordered and unordered)
  • Improve your page with more content

The Image Tag

Display images on your page with the <img> tag:

<img src="path-to-image.jpg" alt="Description of image" width="300">

Key attributes:

  • src - The image file path
  • alt - Text description (important for accessibility)
  • width - Image width in pixels
  • height - Image height in pixels

The Link Tag

Create clickable links with the <a> tag:

<a href="https://google.com">Click here</a>

Link to another page:

<a href="about.html">About Me</a>

Unordered Lists

Use <ul> for bullet-point lists:

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

Ordered Lists

Use <ol> for numbered lists:

<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>

Complete Example

<h1>My Portfolio</h1>
<img src="my-photo.jpg" alt="Me" width="200">
<p>Welcome to my website!</p>

<h2>Skills</h2>
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<h2>Contact</h2>
<a href="mailto:you@email.com">Email me</a>

Your Task for Today

Update your page to include:

  • A profile image (use any image you like)
  • A list of your hobbies or skills
  • A link to an external website
  • At least one link to another page
