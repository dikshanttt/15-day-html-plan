Day 5: Tables and Multi-Page Navigation

Advanced HTML

What You'll Learn Today

HTML Tables

Display data in rows and columns with tables:

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Table Elements

<table> - Container for the table
<tr> - Table row
<th> - Table header cell
<td> - Table data cell
<thead> - Groups header rows
<tbody> - Groups body rows
<tfoot> - Groups footer rows

Project Folder Structure

Organize your files like this:

my-website/
├── index.html
├── about.html
├── contact.html
├── css/
│   └── style.css
├── images/
│   └── my-photo.jpg
└── pages/
     └── recipes.html

Multi-Page Navigation

Create consistent navigation on all pages:

<nav>
  <a href="index.html">Home</a>
  <a href="about.html">About</a>
  <a href="contact.html">Contact</a>
</nav>

Linking Between Pages

From index.html to about.html (same folder):

<a href="about.html">Go to About</a>

From pages/recipes.html back to index.html:

<a href="../index.html">Go Home</a>

The .. means "go up one folder level"

Complete Table Example

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
      <th>Age</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John</td>
      <td>john@email.com</td>
      <td>25</td>
    </tr>
  </tbody>
</table>
