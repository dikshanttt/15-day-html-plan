15-Day Beginner Frontend Development Training

Goal: Complete beginners learn HTML, CSS, and basic JavaScript and build either a Portfolio Website or Recipe Website.

Week 1 — HTML Fundamentals

Day 1: Introduction to HTML

  • How websites work
  • VS Code setup
  • HTML structure
  • Headings, paragraphs, comments

Mini Project: My Introduction Page

Day 2: Images, Links & Lists

  • Images
  • Links
  • Ordered and unordered lists
  • Div and span

Mini Project: Favorite Movie Page

Day 3: Semantic HTML

  • Header
  • Nav
  • Main
  • Section
  • Footer

Mini Project: Personal Profile Page

Day 4: Forms

  • Input fields
  • Labels
  • Radio buttons
  • Checkboxes
  • Textarea

Mini Project: Student Registration Form

Day 5: Tables & Multi-Page Sites

  • Tables
  • Navigation between pages
  • Folder structure

Project: Recipe Website (HTML Only)

Week 2 — CSS Fundamentals

Day 6: CSS Introduction

  • Selectors
  • Colors
  • Fonts
  • Backgrounds

Day 7: Box Model

  • Margin
  • Padding
  • Border
  • Border Radius

Day 8: Flexbox

  • Display Flex
  • Alignment
  • Spacing
  • Responsive layouts

Day 9: Responsive Design

  • Media Queries
  • Mobile Layouts
  • Responsive Navigation

Day 10: Portfolio Styling Project

  • Apply colors
  • Create cards
  • Responsive sections
  • Polish design

Week 3 — JavaScript Fundamentals

Day 11: JavaScript Basics

  • Variables
  • Functions
  • console.log()

Day 12: Click Events

  • onclick
  • addEventListener()
  • Change text
  • Change colors

Day 13: Hover & Input Events

  • mouseover
  • mouseout
  • input events

Project: Live Character Counter

Day 14: DOM Manipulation

  • querySelector()
  • getElementById()
  • textContent
  • classList.toggle()

Projects: Dark Mode, Show/Hide Content

Day 15: Final Project

  • Portfolio Website OR Recipe Website
  • Responsive Design
  • At least 2 JavaScript interactions
  • Clean folder structure

Expected Outcome

  • Create complete multi-page websites
  • Build forms and tables
  • Use Flexbox and Responsive Design
  • Use JavaScript functions and events
  • Create Dark Mode and interactive components
  • Build and present a complete Portfolio or Recipe Website
