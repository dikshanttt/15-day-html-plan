Day 9: Responsive Design

Mobile-First Thinking

What You'll Learn Today

  • What responsive design means
  • Use media queries to adapt to different screen sizes
  • Create mobile-friendly layouts
  • Test your site on different devices

Why Responsive Design?

People view websites on phones, tablets, and desktops. Your site needs to look good on all of them!

  • Mobile users make up 60%+ of web traffic
  • Google ranks mobile-friendly sites higher
  • One responsive site vs. separate mobile site = less work
  • Better user experience = happier users

The Viewport Meta Tag

Always include this in your HTML <head>:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This tells browsers to use the device width and not zoom in.

Media Queries

Media queries let you apply CSS only for certain screen sizes:

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }

  .container {
    padding: 10px;
  }
}

Bootstrap and Responsive Design

Bootstrap includes a responsive grid and helper classes so layouts adapt without writing every media query yourself.

<div class="container">
  <div class="row">
    <div class="col-12 col-md-6">Column 1</div>
    <div class="col-12 col-md-6">Column 2</div>
  </div>
</div>

Common Breakpoints

Mobile: 0-480px
Tablet: 481px-768px
Desktop: 769px-1024px
Large Desktop: 1025px+

Bootstrap Responsive Grid

Bootstrap makes responsive layout easier with its grid system. It automatically adjusts columns for different screen sizes.

<div class="container">
  <div class="row g-3">
    <div class="col-12 col-md-6">Column 1</div>
    <div class="col-12 col-md-6">Column 2</div>
  </div>
</div>

Use classes like col-12, col-md-6, and g-3 for spacing.

Responsive Grid Example

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

@media (max-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 480px) {
  .grid {
    grid-template-columns: 1fr;
  }
}

Responsive Typography

h1 {
  font-size: 36px;
}

@media (max-width: 768px) {
  h1 {
    font-size: 28px;
  }
}

@media (max-width: 480px) {
  h1 {
    font-size: 20px;
  }
}

Your Task for Today

Make your site responsive:

  • Add viewport meta tag to your HTML
  • Create media queries for tablet (768px) and mobile (480px)
  • Adjust padding, margins, and font sizes
  • Test using browser DevTools (F12)
  • Try resizing your browser window
