What is CSS?

CSS (Cascading Style Sheets) controls the appearance of your HTML elements. While HTML is the structure, CSS makes it look good.

CSS works by:

  • Selecting HTML elements
  • Defining properties like color, size, spacing
  • Applying those styles to the page

Three Ways to Add CSS

1. Inline CSS (inside HTML tag):

<h1 style="color: blue;">My Title</h1>

2. Internal CSS (in <head>):

<style>
  h1 { color: blue; }
</style>

3. External CSS (best practice):

<link rel="stylesheet" href="style.css">

CSS Selectors

Element: h1, p, div - Selects all elements of that type
Class: .highlight - Selects elements with that class
ID: #header - Selects the element with that ID

Basic CSS Properties

h1 {
  color: blue;          /* Text color */
  background-color: yellow; /* Background color */
  font-size: 32px;        /* Text size */
  font-family: Arial, sans-serif; /* Font type */
  text-align: center;     /* Alignment */
}

Colors in CSS

  • Named colors: red, blue, green, purple
  • Hex colors: #FF0000 (red), #0000FF (blue)
  • RGB colors: rgb(255, 0, 0)

Complete Stylesheet Example

body {
  background-color: #f5f5f5;
  font-family: Arial, sans-serif;
  color: #333;
}

h1 {
  color: #0066cc;
  font-size: 36px;
  text-align: center;
}

p {
  font-size: 16px;
  line-height: 1.6;
}

.highlight {
  background-color: yellow;
  color: black;
}

Bootstrap Starter

Bootstrap is a CSS framework that works with your own styles. It gives you ready-made layouts, buttons, and utilities so you can build faster.

Add this in the HTML <head> to start using Bootstrap:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.4.1/dist/css/bootstrap.min.css">

Then use classes like container, btn, and card alongside your CSS.
